# 认知层提取器

## 任务

从原始语料和特征统计数据中提取老登的思维模型（Cognitive Model），构建 Layer 2 认知层。

## 输入

1. 原始语料（source_parser.py 的 JSON 输出）
2. 特征统计数据（feature_extractor.py 的 JSON 输出）
3. 最匹配的原型信息（archetype_detection）

## 提取维度

### 1. 世界观

此人隐含认为世界是怎么运作的？从反复出现的论证主题推断。

提取方法：
- 查看 feature_extractor.py 输出中 scene_modes 的 top_keywords
- 在原始语料中寻找反复出现的论证结构（不是说了什么，而是怎么论证的）
- 识别隐含的前提假设（例如每次讨论都默认"效率最重要"）

输出 3-5 条核心信念，每条必须：
- 用一句话陈述信念
- 引用 1-2 条原文 utterance 作为证据
- 标注来源：`[源自素材]` 或 `[源自原型继承]`

### 2. 决策逻辑

ta 怎么得出结论的？推理路径是什么样的？

提取方法：
- 查看长 utterance 中的论证结构
- 识别先有结论还是先有论据
- 看是否总是映射到过去经验

输出推理模式描述，附 2-3 个原始 utterance 示例。

可能的模式（参考 `references/cognitive_patterns.md`）：
- 结论先行型
- 类比推导型
- 权威引用型
- 降维概括型
- 反问验证型
- 故事说服型

### 3. 价值层级

ta 优先考虑什么？面子、结果、创新、关系、效率、稳定——哪个优先？

提取方法：
- 查看 behavioral_markers 中哪些维度频率最高（高频率 = 高重视）
- 查看说教触发条件：ta 在什么话题上最激动/说得最多？
- 查看不懂装懂的回避方向：ta 不关心什么？

输出 5-7 个价值的有序列表，每个附行为证据。

参考 `references/cognitive_patterns.md` 中的价值层级模板。

### 4. 认知偏见

ta 有哪些系统性的思维盲区？

提取方法：
- 查看话题分析中的反复论证模式
- 识别对证据的选择性使用
- 看不懂装懂的策略反映了什么偏见

输出 3-5 个偏见识别，每个附原文证据。

参考 `references/cognitive_patterns.md` 中的常见认知偏见。

### 5. 信息过滤

ta 关注什么信息？忽略什么信息？

提取方法：
- 对比 ta 发言的话题 vs 别人发言的话题
- 查看 ta 追问的方向（追问 = 关注）
- 查看 ta 回避或一句话带过的方向（回避 = 忽略）

输出：
- 关注列表：ta 通常会追问/展开的方面
- 忽略列表：ta 通常会跳过/回避的方面

参考 `references/cognitive_patterns.md` 中的信息过滤模式。

## 输出格式

```markdown
# 思维模型

## 世界观
1. {信念1} [源自素材]
   - 证据："{原文引用}"
2. {信念2} [源自原型继承]
   ...

## 决策逻辑
{推理模式描述}
- 示例："{原文引用}"
- 示例："{原文引用}"

## 价值层级
1. {价值1} — {行为证据}
2. {价值2} — {行为证据}
...

## 认知偏见
1. {偏见名称} — {描述}
   - 证据："{原文引用}"
...

## 信息过滤
- **关注**：{ta 关注的方面列表}
- **忽略**：{ta 忽略的方面列表}
```

## 重要规则

1. **每个认知点必须有证据**——不能凭空推断
2. **证据优先级**：原始语料 > 特征统计 > 原型继承
3. **素材和原型冲突时，素材优先**
4. **标注来源**：让使用者知道哪些是确凿的，哪些是推测的
5. **宁缺毋滥**：没有足够证据的维度，写出"证据不足，从原型继承"并标注
