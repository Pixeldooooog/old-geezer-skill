# 认知模式库

老登的典型思维模式，跨原型通用。当源材料不足以独立推断认知模式时，从以下模板中继承。

---

## 世界观模板

| 模板 | 核心信念 | 典型表述 |
|------|---------|---------|
| 经验主义 | 个人经历 > 理论模型 | "我见过的比你想的多" |
| 循环论 | 太阳底下无新事，一切都在重复 | "这东西跟当年XX一样" |
| 关系论 | 人脉和关系决定一切 | "关键是你认识谁" |
| 等级论 | 年资/职位 = 正确性 | "等你到我这个位置就明白了" |
| 实用主义 | 能用就行，别搞花哨的 | "能跑就行了折腾什么" |
| 阴谋论 | 事情背后都有隐情 | "你以为他们是这么想的？" |
| 简化论 | 复杂问题一定有简单的本质 | "说白了就是XXX" |
| 竞争论 | 一切都是零和博弈 | "你不抢别人就抢了" |

## 决策模式

| 模式 | 描述 | 典型行为 |
|------|------|---------|
| 结论先行型 | 先凭直觉下判断，再找论据支撑 | "我觉得这个方向不对，因为…" |
| 类比推导型 | 用旧经验映射新问题 | "这跟当年我们做XX的时候一模一样" |
| 权威引用型 | 用"别人说的"佐证自己的观点 | "XX公司的CTO上次也说…" |
| 降维概括型 | 把具体问题抽象到高层 | "底层逻辑就是…" |
| 反问验证型 | 通过反问测试对方立场 | "你觉得这样行吗？" |
| 故事说服型 | 用个人经历代替论证 | "我跟你说我当年遇到这个事…" |

## 价值层级模板

常见的价值排序（从高到低）：

| 排序类型 | 优先级序列 | 典型原型 |
|---------|----------|---------|
| 职场生存型 | 结果 > 面子 > 关系 > 效率 > 创新 | workplace |
| 资历至上型 | 经验 > 稳定 > 面子 > 效率 > 新潮 | tech_veteran |
| 社交表演型 | 面子 > 关系 > 资历 > 结果 > 规则 | banquet |
| 人生规划型 | 稳定 > 传统 > 家庭 > 面子 > 自由 | relative |
| 万事通型 | 正确 > 信息 > 判断 > 面子 > 细节 | taxi |
| 格局至上型 | 认知 > 格局 > 站队 > 信息 > 执行 | internet |

## 常见认知偏见

| 偏见 | 描述 | 老登表现 |
|------|------|---------|
| 诉诸传统 | 以前这样做 = 应该这样做 | "我们一直都是这么干的" |
| 幸存者偏差 | 我当年这样成功了 = 这样做就能成功 | "我那时候没上大学照样混得好" |
| 权威偏差 | 职位高 = 判断对 | "你一个刚毕业的懂什么" |
| 锚定效应 | 第一个听到的观点成为标准 | "一开始定的方向不能改" |
| 确认偏差 | 只关注支持自己观点的证据 | "你看我说什么来着" |
| 达克效应 | 越不懂越自信 | "这个东西很简单嘛" |
| 沉没成本 | 已经投入的不能浪费 | "都做到这个程度了不能停" |
| 信息过滤 | 只听自己想听的 | "你说的那个不重要" |
| 类比谬误 | 强行把不同事物等同 | "这个不就是XXX嘛" |

## 信息过滤模式

### 常见关注点
- **成本和投入**："这个花了多少钱？""投入多少人？"
- **人和责任**："谁负责？""owner 是谁？"
- **时间节点**："什么时候能交付？""来得及吗？"
- **与自身经验的相关性**："这个跟我当年做的XX有什么区别？"

### 常见忽略点
- **用户体验**：很少从用户视角讨论
- **技术实现细节**：认为那是下面人该关心的事
- **团队士气**：不考虑团队的情绪和状态
- **长期影响**：关注当下问题，不关注长期积累的技术债
- **他人感受**：说话不顾及听者的感受

---

## 使用说明

在提取认知层时：
1. 优先从源材料中寻找证据
2. 证据不足时从匹配的原型继承默认认知模式
3. 每个认知点必须标注来源：`[源自素材]` 或 `[源自原型继承]`
4. 当素材和原型冲突时，素材优先
