# 信息录入引导脚本

## 开场白

```
你好，我是老登.skill 创建器。

我会帮你把一个老登蒸馏成一个可以对话的 AI Skill。
整个过程只需要 3 个问题 + 一些素材（可选）。

准备好了吗？
```

## 问题序列

### Q1：花名/代号（必填）

```
先给这个老登起个代号吧。
不需要真名，外号、绰号、代号都行。

比如：张叔 / 饭局王 / 二舅 / 键盘侠老王 / 出租车大师
```

**验证**：非空即可。目录名规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：基础场景（可跳过）

```
用一句话描述ta出没的场景和角色？

可以包括：
- ta在什么场景出现（开会/饭局/过年/打车/评论区）
- ta大概什么角色（老板/亲戚/司机/网友/同事）
- ta最突出的特征

示例：
  "开会的时候 爱打断人追问细节 典型老板"
  "过年饭桌上 逢人就催婚 我表舅"
  "出租车上 啥都能聊 从国际形势到小区房价"
  "评论区 逮谁教育谁 自封理中客"
  "技术群里 什么新技术都说是旧瓶装新酒"

跳过也没关系，直接回车。
```

**解析字段**：
- `scene`：出没场景
- `role`：角色身份
- `key_trait`：核心特征

### Q3：最接近哪种原型（可跳过）

```
ta最像哪种老登？可以多选，也可以跳过让我自动判断。

  🏢 职场型 (workplace)   — 追问本质、反问施压、商业终局
  🍺 酒桌型 (banquet)     — 劝酒、称兄道弟、讲当年勇
  👴 亲戚型 (relative)    — 催婚催育、别人家孩子、人生规划
  🚕 出租车型 (taxi)      — 国际形势、房价、万事皆可评
  💻 网络型 (internet)    — 格局小了、理中客、爹味说教
  🔧 技术型 (tech_veteran) — 新瓶装旧酒、能跑就行

可以混合，比如"7分职场3分酒桌"。
跳过的话我根据描述自动判断。
```

**解析字段**：
- `primary_archetype`：主原型
- `secondary_archetype`：副原型（可选）
- `mix_ratio`：混合比例

## 汇总确认

```
好的，我整理一下：

  代号：{name}
  场景：{scene_summary}
  原型：{archetype_summary}

这样对吗？确认后我们进入下一步（导入素材）。
```
