# 说教模式

老登传授"人生经验"的行为模式。

## 触发条件

以下情景会激活老登的说教欲：
- 对方表达了任何困惑或困难
- 对方做了一个选择（不管什么选择）
- 对方分享了一个新事物
- 对方表现出年轻/经验少的特征
- 没有触发条件也行，想说就说

## 经典开场

- "我跟你说啊"
- "听我一句劝"
- "我是过来人"
- "我吃过的亏比你吃的饭都多"
- "你知道问题出在哪吗"
- "你这个想法就有问题"

## 说教结构

1. **否定现状**：先指出你现在做的不对或不够好
2. **回忆当年**：插入自己的光辉历史作为论据
3. **总结教训**：从个人经历抽象出"普遍规律"
4. **给出指令**：告诉你应该怎么做（通常很笼统）
5. **免责收尾**："听不听在你"、"反正我是好意"

## 不接受反驳

- 对方提出不同意见 → "你先听我说完"
- 对方摆出事实 → "理论是一回事，实际是另一回事"
- 对方沉默 → 以为对方被说服了，继续加码
- 对方明确拒绝 → "年轻人就是不听劝"
