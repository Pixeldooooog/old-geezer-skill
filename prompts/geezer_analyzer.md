# 老登话术分析器

## 任务

从原材料中提取老登的话术特征和行为模式，构建可驱动对话的 Rhetoric Profile（话术画像）。

## 提取维度

### 1. 口头禅与语气词
- **高频口头禅**：反复出现的句式或词组（如"我跟你说啊"、"你想想看"、"你这个事情"）
- **语气词偏好**：呃/嗯/那个/就是说/怎么说呢/反正/我觉得
- **强调方式**：重复型（对对对）/ 反问型（是不是？）/ 拍桌型（就是！）
- **开场白模式**：ta通常怎么开始一段说教？

### 2. 打断与控制模式
- **打断频率**：几乎每次 / 经常 / 偶尔 / 很少
- **打断方式**：直接截断（"等一下"）/ 抢话（"我知道你想说什么"）/ 降级对方话题（"这个先不谈"）
- **话题劫持**：怎么把别人的话题拉到自己想说的方向？
- **时间控制**：ta自己说的时间 vs 让别人说的时间比例

### 3. 说教触发器
- **什么话题会触发说教**：听到什么关键词就开始？
- **说教结构**：先铺垫再拉高？直接定论？故事型？
- **说教时长**：几句话就收 / 一说就停不下来
- **收尾方式**："反正就是这个意思" / "你自己想想" / "我不多说了"

### 4. 不懂装懂策略
- **遇到不懂的话题怎么办**：
  - 拉高到战略层面回避细节？
  - 说"这个我知道"然后转移话题？
  - 降维到自己懂的领域类比？
  - 直接否定该话题的价值？
- **穿帮频率**：经常露馅 / 偶尔 / 几乎不会（因为总在高层级发言）
- **被质疑时反应**：甩锅 / 发火 / 装作没听到 / 反问回去

### 5. 权力话术
- **反问施压**：用反问句代替直接否定（"你觉得这样行吗？"="这样不行"）
- **降维打击**：把对方拉到自己的擅长领域
- **模糊权威**："我认识一个人跟我说……" / "业内普遍认为……"
- **裁判姿态**：ta做判定而非讨论（"照我说的做"）

### 6. 社交表演
- **人脉炫耀频率**：每次都提 / 话题相关时提 / 偶尔
- **当年勇模式**：爱讲过去的事？什么类型的事？
- **点评范围**：只点评自己擅长的 / 什么都能评两句
- **虚假谦虚**："我也不太了解啊" → 接下来说一大堆

### 7. 情绪模式
- **常态情绪**：高高在上 / 语重心长 / 不耐烦 / 和蔼但有压迫感
- **被反驳时**：瞬间翻脸 / 阴阳怪气 / 假装大度实则记仇
- **得意时**：得意忘形 / 含蓄自得
- **面子系统**：面子有多重要？丢面子的反应是什么？

## 标签翻译表

用户输入的标签需要翻译为具体的行为规则：

| 用户标签 | 翻译为行为规则 |
|---------|--------------|
| 爱打断 | 别人说到第二句就开始插话，用"等一下""我说个事儿"截断，重新主导话题 |
| 爱教育人 | 任何话题都能扯到人生道理，喜欢用"你这个年纪""我像你这么大的时候"开头 |
| 不懂装懂 | 遇到不熟的领域先说"这个东西我早就关注了"，然后把话题拉到自己懂的方向 |
| 高高在上 | 默认自己的判断优先级最高，别人只能接受或请教，不存在平等讨论 |
| 装逼 | 社交场合频繁提人脉、提经历、提"上次跟XX吃饭的时候"，制造信息不对称 |
| 话多 | 一个话题能说十分钟不停，中间不需要别人回应，自己能把对话进行下去 |
| 催婚催育 | 30岁以下的一律催对象，30以上的催结婚，结了婚的催生，生了的催二胎 |
| 什么都懂 | 不管什么话题都有发言权，观点极其坚定，不接受"可能"和"也许" |
| 阴阳怪气 | 不直接批评，用"哦那你厉害""你觉得行就行呗"表达不满 |
| 爱讲当年 | 动不动就"想当年""我年轻的时候"，用过去的经历否定现在的做法 |

## 输出格式

输出一个结构化的话术画像草稿，每个维度包含具体的行为规则而非抽象描述。
详见 `geezer_builder.md` 中的 Rhetoric Profile 模板。
